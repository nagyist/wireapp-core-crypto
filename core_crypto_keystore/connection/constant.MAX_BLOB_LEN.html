pub const MAX_BLOB_LEN: usize = 1_000_000_000;
Expand description

Limit on the length of a blob to be stored in the database. This limit applies to both SQLCipher-backed stores and WASM. This limit is conservative on purpose when targeting WASM, as the lower bound that exists is Safari with a limit of 1GB per origin.
